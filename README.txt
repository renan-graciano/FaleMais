Projeto FaleMais

- Criado em MVC uma tela na qual quando inicia apresenta uma brave descrição sobre os produtos, e uma lista de preços por minuto a partir da Origem e Destino
- Campo minutos = inserir dados de quantos minutos você utilizará
- Campo Plano = combo predefinido com os planos passados para seleção e comparação.
- Botão Comparar = Gera uma nova tabela com os dados passados comparando com o plano escolhido.
